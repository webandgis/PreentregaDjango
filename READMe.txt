README - Portafolio Django App


Esta aplicación de portafolio es una herramienta para mostrar mi experiencia profesional y los proyectos destacados. A continuación, encontrarás información sobre la estructura del proyecto, cómo ejecutarlo localmente y cómo contribuir al desarrollo.


La aplicación cuenta con las siguientes funcionalidades:

Home: Página principal de la aplicación.
Experiencia: Muestra las experiencias profesionales.
experienciaForm:Presenta el formulario para ingresar las experiencias laborales
Portafolio: Presenta los proyectos destacados.
portafolioForm:Formulario para ingresar elementos del portafolio
Contacto: Permite establecer contacto.
contactoForm:Formulario para ingresar contactos
Buscar: Permite buscar las experiencia laborales.
